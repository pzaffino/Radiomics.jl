Installation Guide for Radiomics.jl

Installation

To install Radiomics.jl, simply run the following in Julia:

import Pkg
Pkg.add("Radiomics")

You can install Radiomics.jl on:

  • Your local machine (desktop/laptop).
  • Google Colab (with Julia kernel).

Notes

Once installed, you can start using Radiomics.jl functions such as get_gldm_features, get_ngtdm_features, and other radiomics feature extraction tools directly in Julia.
