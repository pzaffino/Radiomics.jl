<p align="center">
  <strong>An open-source Julia library for extracting radiomics features from medical images.</strong>
</p>

<p align="center">
   <em>Radiomics.jl is in an early stage and under active development</em> ⚠️
</p>

---

## Implemented Features

- **First-order features**
- **2D shape features**
- **3D shape features**
- **Texture matrices:**
  - GLCM
  - GLDM
  - GLRLM
  - GLSZM
  - NGTDM
---
