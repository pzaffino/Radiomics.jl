Gray Level Run Length Matrix (GLRLM) feature extraction in Julia

GLRLM Feature Extraction

This page documents the main functions for computing 2D and 3D GLRLM features, which quantify the length of consecutive runs of voxels with the same intensity.

Key Features

  • Short Run Emphasis (SRE): measures the distribution of short runs; higher values indicate more short runs and finer textures.
  • Long Run Emphasis (LRE): measures the distribution of long runs; higher values indicate coarser textures.
  • Gray Level Non-Uniformity (GLN): measures similarity of gray level values; lower values indicate more uniform gray levels.
  • Gray Level Non-Uniformity Normalized (GLNN): normalized version of GLN.
  • Run Length Non-Uniformity (RLN): measures similarity of run lengths; lower values indicate more uniform run lengths.
  • Run Length Non-Uniformity Normalized (RLNN): normalized version of RLN.
  • Run Percentage (RP): measures the distribution of runs relative to the total number of voxels; indicates texture density.
  • Gray Level Variance (GLV): measures variance in gray level intensities.
  • Run Variance (RV): measures variance in run lengths.
  • Run Entropy (RE): measures disorder or randomness in run lengths and gray levels.
  • Low Gray Level Run Emphasis (LGLRE): emphasizes runs with low gray levels.
  • High Gray Level Run Emphasis (HGLRE): emphasizes runs with high gray levels.
  • Short Run Low Gray Level Emphasis (SRLGE): emphasizes short runs with low gray levels.
  • Short Run High Gray Level Emphasis (SRHGE): emphasizes short runs with high gray levels.
  • Long Run Low Gray Level Emphasis (LRLGE): emphasizes long runs with low gray levels.
  • Long Run High Gray Level Emphasis (LRHGE): emphasizes long runs with high gray levels.

Notes

GLRLM features are especially useful for capturing coarse and fine textures in medical images. These metrics are computed over multiple directions and averaged to obtain robust values for a region of interest.
