Neighbouring Gray Tone Difference Matrix (NGTDM) feature extraction in Julia

NGTDM Feature Extraction

This page documents the main functions for computing NGTDM features, which quantify differences in gray levels between a voxel and its neighborhood.

Key Features

  • Coarseness: measures the spatial rate of change in intensity; higher values indicate coarser textures.
  • Contrast: quantifies the intensity contrast between a voxel and its neighbors; higher values indicate larger differences.
  • Busyness: measures rapid changes in intensity between neighboring voxels; higher values indicate more heterogeneous textures.
  • Complexity: evaluates local intensity variations considering differences among all gray levels; higher values indicate more complex textures.
  • Strength: measures the perceptual strength of the texture patterns; higher values indicate more prominent structures.

Usage

# Using a fixed number of bins (bin_width calculated automatically)
features = get_ngtdm_features(img, mask, spacing, n_bins=64)

# Using a fixed bin width (number of bins calculated automatically)
features = get_ngtdm_features(img, mask, spacing, bin_width=25.0f0)

# Default (32 bins)
features = get_ngtdm_features(img, mask, spacing)

Calculation Steps

  1. Discretize the image using n_bins or bin_width.
  2. Compute the NGTDM matrix using calculate_ngtdm_matrix, which considers the neighborhood of each voxel.
  3. Compute coefficients with calculate_ngtdm_coefficients (including probabilities p_i and differences s_i).
  4. Calculate each feature using its specific formula, e.g., coarseness, contrast, busyness, complexity, strength.

Notes

NGTDM features are particularly useful in medical imaging for capturing local texture patterns and assessing heterogeneity within a region of interest.
